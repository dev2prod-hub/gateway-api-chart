Gateway API Helm Chart

Kubernetes Gateway API — the successor to Ingress. Install via Helm from https://charts.cdnn.host/.

Quick Install

helm repo add dev2prod https://charts.cdnn.host/
helm repo update
helm install my-gateway dev2prod/gateway-api
helm install routes dev2prod/gateway-api-routes

Charts

  • gateway-api — GatewayClass, Gateway, CRDs (v1.4.1)
  • gateway-api-routes — HTTPRoute, GRPCRoute, TCPRoute, UDPRoute
